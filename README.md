<h2>How It's Made:</h2>


<p>Tech used: Express.js & MongoDB</p>

<p>I created Hicks Landscaping Services using the Express.js frontend framework and the MongoDB NoSQL database. These frameworks allowed me to create a user-friendly and scalable website with a high degree of maintainability. </p>
